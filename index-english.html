Welcome to Box CSS!

Box CSS provides you with a 12-column grid where you can adapt your design and distribute your content.

Start with the mobile design (small) and make modifications for larger screens with the styles you need.

If you just want to use the columns for a desktop site, using .small-# will be sufficient.

Download: Box CSS and JS file for Media Query compatibility with IE8

Let's go over the basics

Prefixes

Small: Maximum screen size of 640px

Medium: Screen size from 641px to 1024px

Large: Screen size from 1025px to 1440px

XLarge: Screen size from 1441px to 1920px

XXLarge: Screen size above 1921px

Grid

.medium-6 represents 6 columns starting from medium screens
.medium-6 on small screens will be a single column
.large-6 represents 6 columns starting from large screens
.large-2 represents 2 columns
.large-4 represents 4 columns
.show-for-large Visible on large screens or larger
.hide-for-large Hidden on large screens or larger
.show-for-small-only Visible only on small screens

Quick Styles

.text-left Left alignment

.text-center Center alignment

.text-right Right alignment

.text-justify Justified alignment

.float-left
Floats left
.float-right
Floats right

Box CSS was inspired by Foundation but is simpler and more search engine friendly. Compatible with all modern browsers and IE8+
